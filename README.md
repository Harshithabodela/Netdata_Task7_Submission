# Netdata System Monitoring

## 🔧 Setup
Used Docker to install and run Netdata:
```bash
docker run -d --name=netdata -p 19999:19999 --cap-add SYS_PTRACE --security-opt apparmor=unconfined netdata/netdata
```

Access Dashboard:  
http://localhost:19999
